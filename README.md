## 📖 About
Gradixy is a powerful, lightweight **CSS Gradient Generator** designed to streamline the developer workflow. It provides a sleek visual interface to craft complex backgrounds and instantly exports production-ready code.

* **Linear, Radial, and Conic** support.
* Instant **CSS & Tailwind CSS** export.
* Zero dependencies – built with pure Vanilla JS.

## 🚀 Features
* ✨ **Live Preview:** See changes in real-time.
* 🎨 **Multi-Type Support:** Generate Linear, Radial, and Conic gradients.
* ⚡ **One-Click Copy:** Get CSS or Tailwind classes instantly.
* 🎲 **Randomizer:** Generate fresh inspiration with one click.
* 🛠️ **Granular Control:** Adjust angles, color stops, and positions.
* 📱 **Fully Responsive:** Works on desktop and mobile browsers.

## 🛠️ Technologies

## ⚙️ Installation
To run this project locally, no build tools are required.

Clone the repository
```
Bash
git clone [https://github.com/HAMZFLST-TECH/Gradixy.git](https://github.com/HAMZFLST-TECH/Gradixy.git)
```
Navigate to the folder

```Bash
cd Gradixy
```
## Launch
Simply open index.html in your preferred browser.
